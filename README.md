 ## This is a React Application that calls the Google Books API based on a search
 
 ## USAGE

To use this app you must first have an API key associated with Google Books. Info on how to obtain one is at https://developers.google.com/books/docs/v1/using#APIKey
 Create an environment variable in a .env file off the root with a Google Books API key called

 VITE_BOOKS_API_KEY=\<api key\>
 

Before running the app

To run the app after cloning this repo, run
npm install
npm run dev
